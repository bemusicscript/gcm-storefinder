# gcm-storefinder

This is a map generated by crawling data from [https://location.am-all.net](https://location.am-all.net).  
It allows you to easily find arcades where all supported machines are installed together.

[https://location.am-all.net](https://location.am-all.net)のデータをクローリングして作成した店舗マップです。  
すべての対応機種が同じ場所に設置されているゲーセンを簡単に探すことができます。

🗓 **Update (2024-11-02):** Now supports global store locations. (海外店舗対応済み)
